# Deskripsi
Aplikasi CRUD data guru, kelas dan murid

# Instalation
Install node atau nvm
run di terminal
```sh
npm i
```
untuk menginstall library

# Run API
```sh
npm start 
```
atau
```sh
npm run dev
```
